Introduction

单因子方差分析

学习目标

  • 目标
    • 掌握单因子方差分析的流程和原理
    • 知道事后检验的作用

目标: 用来判断由一个名义变量分割的多个总体的均值是不是相等

方差分析(analysis of variance)，简写为ANOVA，指的是利用对多个样本的方差的分析，得出总体均值是否相等的判定。

先来个例子

为了检验某小学六年级教学质量的差异，从该小学六年级的三个班级中分别选取一定数量的学生，分成三个组（三个样本），对他们期末考试的平均分进行统计分析。如果实验显示每个每组的均值相同，即三个班期末考试的成绩差异不大，则表明该小学六年级不同班级的教学质量没有差异，

每个样本组的平均分分别为μ1,μ2,μ3\mu_1,\mu_2,\mu_3μ​1​​,μ​2​​,μ​3​​，方差分别为σ12,σ22,σ32\sigma_1^2,\sigma_2^2,\sigma_3^2σ​1​2​​,σ​2​2​​,σ​3​2​​

给出零假设: H0:μ1=μ2=μ3H_0:\mu_1=\mu_2=\mu_3H​0​​:μ​1​​=μ​2​​=μ​3​​

备择假设H1H_1H​1​​：样本组的均值不全相等(μ1≠μ2\mu_1\neq\mu_2μ​1​​≠μ​2​​或μ2≠μ3\mu_2\neq\mu_3μ​2​​≠μ​3​​或μ1≠μ3\mu_1\neq\mu_3μ​1​​≠μ​3​​)

方差分析将会依据观测数据判定假设是否成立。

1.假定条件

  1. 每个样本的值服从正态分布
  2. 每个样本的方差σ2\sigma^2σ​2​​相同
  3. 每个样本中的个体相互独立

2.统计量—F值

  • 误差平方和SSESSESSE

    误差平方和就是残差,即由其他位置因素引起的误差,值为每一组的误差和

    ​ ∑i=1k∑j=1ni(Xij−Xi¯)2\sum_{i=1}^{k}{\sum_{j=1}^{n_i}{(X_{ij}-\bar{X_i})^2}}∑​i=1​k​​∑​j=1​n​i​​​​(X​ij​​−​X​i​​​¯​​)​2​​

    k为组数

    nin_in​i​​为第i组的数据数量

    XijX_{ij}X​ij​​为第i组第j个数据

    Xi¯\bar{X_i}​X​i​​​¯​​为第i组的均值

    计算方法:

    • 将各组中的每个值减去该组均值Xij−Xi¯ X_{ij}-\bar{X_i}X​ij​​−​X​i​​​¯​​(离差)
    • 将将该组中这些离差取平方(Xij−Xi¯)2(X_{ij}-\bar{X_i})^2(X​ij​​−​X​i​​​¯​​)​2​​
    • 将该组中这些离差平方求和∑j=1ni(Xij−Xi¯)2\sum_{j=1}^{n_i}{(X_{ij}-\bar{X_i})^2}∑​j=1​n​i​​​​(X​ij​​−​X​i​​​¯​​)​2​​
    • 将所有组的离差平方和求和∑i=1k∑j=1ni(Xij−Xi¯)2\sum_{i=1}^{k}{\sum_{j=1}^{n_i}{(X_{ij}-\bar{X_i})^2}}∑​i=1​k​​∑​j=1​n​i​​​​(X​ij​​−​X​i​​​¯​​)​2​​

  • 组间误差平方和SSBSSBSSB

    组间误差平方和就是由分组变量引起的误差,是分组变量能够解释的误差,体现在各组均值的差异上.计算公式:

    ​ ∑i=1kni(Xi¯−X¯)2\sum_{i=1}^{k}{n_i(\bar{X_{i}}-\bar{X})^2}∑​i=1​k​​n​i​​(​X​i​​​¯​​−​X​¯​​)​2​​

    k为组数

    nin_in​i​​为第i组的数据数量

    Xi¯\bar{X_i}​X​i​​​¯​​为第i组的均值

    X¯\bar{X}​X​¯​​为总均值

  • 自由度

    • 分子:k-1
    • 分母:n-k

第三步:计算误差平方和(SSeSS_eSS​e​​)

首先计算各单个取值与所在组均值之间的离差平方,将三个平方和相加得到SSe=10.8SS_e=10.8SS​e​​=10.8

第四步:计算组间平方和(SSbSS_bSS​b​​)

第五步:计算均方误(MSeMS_eMS​e​​)

将误差平方和(SSeSS_eSS​e​​)除以误差自由度(N−KN-KN−K): MSe=SSeN−K=10.815−3=0.90\Large MS_e=\frac{SS_e}{N-K}=\frac{10.8}{15-3}=0.90MS​e​​=​N−K​​SS​e​​​​=​15−3​​10.8​​=0.90

第六步:计算组间均方(MSbMS_bMS​b​​)

将组间平方和(SSbSS_bSS​b​​)除以组间自由度(K−1K-1K−1): MSb=SSbK−1=8.903−1=4.45\Large MS_b=\frac{SS_b}{K-1}=\frac{8.90}{3-1}=4.45MS​b​​=​K−1​​SS​b​​​​=​3−1​​8.90​​=4.45

第七步:计算FFF比率

F=MSbMSe=4.450.90=4.94\Large F=\frac{MS_b}{MS_e}=\frac{4.45}{0.90}=4.94F=​MS​e​​​​MS​b​​​​=​0.90​​4.45​​=4.94

第八步:查表得到FFF临界值

分子自由度为2,分母自由度为12,查表得到FFF临界值为3.88(α=0.05\alpha=0.05α=0.05)

第九步:判定FFF值是否统计显著

做出结论,认为统计结果是显著的,意味着各组均值不完全相等,就是说至少有两组的均值不相等

代码手动实现:

  • 获取数据

    #加载数据
x=np.array([5,5,4,4,3,5,4,4,3,3,4,3,2,2,1])

  • 计算各组均值

    #计算各组均值

    #计算总均值
mean=data['x'].mean()

  • 计算总误差SSTSSTSST

    总误差就是数据整体的方差*n : ∑(Xi−X¯)2\sum{(X_i-\bar{X})^2}∑(X​i​​−​X​¯​​)​2​​

    #计算总误差SST
SST=np.sum(np.square(data['x']-mean))
SST#19.733333333333334

  • 计算残差(误差平方和)SSESSESSE

    ∑i=1k∑j=1ni(Xij−Xi¯)2\sum_{i=1}^{k}{\sum_{j=1}^{n_i}{(X_{ij}-\bar{X_i})^2}}∑​i=1​k​​∑​j=1​n​i​​​​(X​ij​​−​X​i​​​¯​​)​2​​

    #计算残差(误差平方和)SSE
SSE_5=np.sum(np.square(data[data['age']==5]['x']-means.loc[5,'mu']))
SSE_8=np.sum(np.square(data[data['age']==8]['x']-means.loc[8,'mu']))
SSE_12=np.sum(np.square(data[data['age']==12]['x']-means.loc[12,'mu']))
SSE=SSE_5+SSE_8+SSE_12
SSE#10.8

  • 计算年龄带来的误差(组间平方和)SSBSSBSSB

    ​ ∑i=1kni(Xi¯−X¯)2\sum_{i=1}^{k}{n_i(\bar{X_{i}}-\bar{X})^2}∑​i=1​k​​n​i​​(​X​i​​​¯​​−​X​¯​​)​2​​

    #计算年龄带来的误差(组间平方和)SSB
SSB_5=5*(means.loc[5,'mu']-mean)**2
SSB_8=5*(means.loc[8,'mu']-mean)**2
SSB_12=5*(means.loc[12,'mu']-mean)**2
SSB=SSB_5+SSB_8+SSB_12

statsmodels实现:

能够输出方差分析表,更优雅

#statsmodels实现
from statsmodels.formula.api import ols
from statsmodels.stats.anova import anova_lm
#分组字段必须是字符串类型
data.loc[:,'age']=data['age'].astype(np.str_)

#最小二乘法拟合
model = ols('x ~ age',data).fit()
#到处方差分析结果
anovat = anova_lm(model)
anovat

4. 事后检验

在判定组均值之间有显著差异后,仍有一些问题悬而未决.这个结果只能表明至少有两个组之间的均值有显著差异,但没有说明究竟哪几个组均值显著不同.我们必须进行事后检验

TukeyHSDTukey HSDTukeyHSD—TukeyTukeyTukey检验

查询对应的统计量(ttt分布)表来判定两个组均值是否显著差异

继续上面的案例

结论:组1和组3有显著差异

statsmodels实现:

from statsmodels.stats.multicomp import MultiComparison
mc = MultiComparison(data['x'],data['age'])
tukey_result = mc.tukeyhsd(alpha = 0.05)
print(tukey_result)

从结果中可以看出,5-12两组间拒绝原假设,有明显差异.其余两组间无明显差异

5.小结

  • 单因子方差分析:用来判断由一个名义变量分割的多个总体的均值是不是相等
  • 统计量:
    • F=MSbMSe\Large F=\frac{MS_b}{MS_e}F=​MS​e​​​​MS​b​​​​
      • 意义是分组变量解释的误差/不可解释误差 的比例,该值越大,说明分组变量影响越大
    • SSTSSTSST:总误差
    • SSBSSB SSB:组间误差(分组变量可解释的误差)
    • SSESSE SSE:残差(不可解释的误差)
  • api:
    • scipy: scipy.stats.f_oneway(*args)
    • statsmodels:
      • ols('x ~ age',data).fit()最小二乘拟合
      • anova_lm(model)输出方差分析表
  • 事后检验:
    • 作用:找出有差异的组
